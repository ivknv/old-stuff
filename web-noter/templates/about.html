I'm a newbie in programming.
I prefer Python+Django.

What is Noter?

Noter is a note keeping program.
Notes can be accessed from this site or directly from database.

Why?

Because there's no perfect tool for keeping notes and Noter is not an exception.
But Noter is created with soul, it has easier UI, it has evolved from a tiny command line script to a site (and still can be used from a command line), it's open source and it's written in Python.

Noter vs. monsters like Evernote

I'm not trying to say that monsters like Evernote sucks.
I'm just trying to say Noter is much lighter and it's definetly not a monster.
It's easy to use, fully free, simple and new.
And what about our almost favorite monsters? Here's the list:

  1. Too big
  2. With limits of free usage
  3. Too Complicated
And even if you're still not impressed I still like Noter because I wrote it by myself, with soul.

History

Actually, It's weird to write history for a common app. But who cares at all?
First I've got an idea to make my own note keeping app after I've got my phone (Huawei U8110) broken.
I was always keeping notes before, that was a part of my life. I used ColorNote for Android.
But there was so many ideas... I had to write them somewhere.
Before it was my phone but what should I do when I have my phone broken?
So I decided to create Noter. After few months I also decided to write this site.
And now Noter is one of my favorite projects. I think I will keep it evolving for long time.
